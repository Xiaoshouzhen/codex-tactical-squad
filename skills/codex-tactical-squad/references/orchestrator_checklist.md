# Orchestrator Checklist

1. Restate the task and decide whether the triad should be used.
2. Create a task intake.
3. Ask the planner for a round contract.
4. Tighten ambiguous scope or acceptance criteria before execution.
5. Send the approved contract to the executor with explicit ownership.
6. Send the task, contract, and executor report to the reviewer.
7. Send the review result to the planner.
8. Accept only if there are no material findings or residual risk is explicitly accepted.
9. If needed, run a narrow follow-up round instead of reopening the whole task.
